## Part 5 : Streamlit Dashboard
### How to run:
1. create and activate an env
```bash
python3 -m venv venv
source venv/bin/activate  #mac
```
2. install needed packages
```bash
pip install streamlit pandas matplotlib sqlite3
```
3. run it 
```bash
streamlit run part5.py
```
